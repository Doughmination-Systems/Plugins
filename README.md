# Doughmination Plugins

Minecraft server Plugins build on the Doughmination Framework.

## Plugins

The list of plugins include:
- DoughAPI
- Doughcord
- JailedDough
- DoughEndStats
- Plural
- WingSync

## Quick Start

```bash
# Build all
./gradlew buildAll

# Build specific plugin
./gradlew :pluginname:shadowJar
```

**Installation:** Download from [Doughmination Modding](https://modding.doughmination.win)
